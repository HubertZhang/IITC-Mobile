Webview Console

A console is added for development and debugging plugins.

Script Viewer

A script viewer with code highlighting is added. Just select a downloaded script to check its content.

Now you can search plugins by its name, category, and descriptions. Just scroll down the plugin list if it doesn't appear.

Custom UserAgent

Allow customizing UserAgent of the browser. You can either choose a pre-defined UserAgent or use your own one.
